# Restaurant Business Performance Analysis

## Data Exploration & Initial Assessment

### Objective

The purpose of this analysis is to explore the raw datasets, understand their structure, assess data quality, and identify any preparation required before performing business analysis.

# Initial Data Assessment Summary

## Findings

- The datasets contain transactional order data and menu information.
- Data types were reviewed and potential formatting changes identified.
- Missing values and duplicate records were assessed.
- Relationships between tables were identified through item identifiers.

## Next Steps

The next stage will focus on data cleaning and transformation to prepare the datasets for SQL analysis and dashboard development.
