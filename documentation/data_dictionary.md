# Data Dictionary

## Order Details Table

| Column | Description |
|---|---|
| order_id | Unique identifier for each order |
| order_date | Date the order was placed |
| item_id | Unique menu item identifier |
| quantity | Number of items purchased |


## Menu Items Table

| Column | Description |
|---|---|
| menu_item_id | Unique product identifier |
| item_name | Name of menu item |
| category | Food category |
| price | Selling price |
